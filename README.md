# Clojure M-Pesa Wrapper

A clojure wrapper around mpesa daraja api

Work in progress, feel free to contribute.

Supported endpoints,
1. Authentication 
2. B2C API
3. B2B API
4. Check balance
5. C2B API
6. Lipa na Mpesa
7. Lipa na M-Pesa Online
7. Reversal 

Looking to include, 
1. Transaction Status
