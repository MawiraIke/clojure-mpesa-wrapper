# Clojure M-Pesa Wrapper

![M-Pesa](images/mpesa.png)

A clojure wrapper around mpesa daraja api


Supported endpoints,
1. Authentication 
2. B2C API
3. B2B API
4. Check balance
5. C2B API
6. Lipa na Mpesa
7. Lipa na M-Pesa Online
8. Reversal 
9. Transaction Status
